# AssetVault

AssetVault is a VRChat and Unity asset manager designed to help creators organize assets like avatars, clothing, props, textures, shaders, and packages.

## Features

• Drag & drop asset importing  
• Automatic file categorization  
• VRChat asset organization  
• Asset preview images  
• Favorites system  
• Rating system  
• Tag system  
• Fast asset library browsing

## Categories

- Avatars
- Clothing
- Accessories
- Hair
- Props
- World Assets
- Textures
- Shaders
- Animations
- Audio
- Unity Packages
- NSFW
- Other
